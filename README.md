안녕하세요! 2019년 이후로 첫 TIL 저장소 입니다.
이번 TIL은 지난번 TIL(비공개로 전환)과 달리 계획을 이슈에 등록하여 진행 결과 또한 이슈에 매일 기록할 것 입니다.

그리고 공부한 것을 간단히 요약해서 주말마다 업로드 할 것 입니다. 형식은 마크다운으로 진행할것입니다.

이번 TIL에 기록하는 내용은 크게 3가지로 나누어집니다.
- 경찰 간부 시험(사이버)
- 학업
- 개발

특히 많은 오타와 오번역을 유발과 정보의 잘못된 서술들을 기록하였을때 편하게 **issue**를 보내주시면 감사하겠습니다.
